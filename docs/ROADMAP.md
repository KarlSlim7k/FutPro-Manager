# UI/UX Audit Roadmap

## Fase 1
Auditoría inicial UI/UX, contraste, CTAs, `focus-visible`, favicon/metadata y correcciones críticas.

## Fase 2
Consistencia visual estructural con componentes reutilizables:
- `TextLink`
- `Eyebrow`
- `StatusBadge`
- `EmptyState`
- `SectionHeader`
- `PageHeader`
- `MetricCard`
- `FormSectionCard`

## Fase 3
Pulido visual/UX del dashboard, responsive, `ToolbarActions`, accesibilidad básica y consistencia de acciones.

## Fase 4
Auditoría pre-MVP, semántica, links externos, controles de formulario, densidad mobile y documentación del sistema visual.

## Fase 5
QA visual final, revisión cross-browser/manual, checklist de release y preparación para pruebas con usuarios reales.
